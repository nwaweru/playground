# Tasks Manager - Demo of React with Laravel

Based on [this post](https://blog.pusher.com/react-laravel-application "Blog Post") from Pusher.

## To try this out:
1. `git clone [git_url]`
1. `cd [project_name]`
1. `composer install && npm i && npm run dev`
1. Update your `.env`
1. `php artisan serve`
